# ATS SQL Project

## 📌 Project Overview

The ATS SQL Project is a MySQL-based Applicant Tracking System (ATS) database designed to manage the complete recruitment process.

This project demonstrates SQL concepts including:

- Database Design
- Table Creation
- Relationships
- Constraints
- CRUD Operations
- Joins
- Aggregate Functions
- Subqueries
- Views
- Stored Procedures
- Triggers
- Transactions

---

## 🚀 How to Run

1. Open MySQL Workbench.
2. Execute `01_Create_Database.sql`.
3. Execute `02_Create_Tables.sql`.
4. Execute `03_Insert_Data.sql`.
5. Execute the remaining SQL files in order.
